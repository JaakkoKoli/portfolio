    Hamiltonian Monte Carlo demo

    [code] [app]

    An interactive demonstration of the Hamiltonian Monte Carlo algorithm using JavaScript and Gatsby. The demo lets the user change different parameters such as the amount of Leapfrog integrator steps and immediately see how it affects the sampling which can provide valuable insights when trying to understand how this method works and what the different parameters do.

    Technologies/skills used: JavaScript, Gatsby, React, statistics
