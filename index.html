    Hamiltonian Monte Carlo demo

    [code] [app]

    This project is an interactive web application built with JavaScript and Gatsby, designed to provide an intuitive and visual understanding of the Hamiltonian Monte Carlo (HMC) algorithm—a powerful method used in statistical mechanics and Bayesian inference. The app focuses on demonstrating how key parameters of the HMC algorithm, such as initial position, leapfrog steps, and leapfrog step sizes, influence the sampling process and final results.
    Users can dynamically adjust these parameters and observe in real-time how the algorithm explores the target probability distribution. The app features interactive visualizations, including particle trajectories and energy landscapes, to help users grasp the underlying mechanics of HMC. This hands-on approach makes complex concepts accessible to both beginners and experienced practitioners in machine learning, statistics, and data science.

    Technologies/skills used: JavaScript, Gatsby, React, statistics
